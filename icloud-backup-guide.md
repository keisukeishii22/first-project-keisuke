# iCloudバックアップ ガイド

## iCloudバックアップを有効にする

1. **設定** → 上部の自分の名前（Apple ID）をタップ
2. **iCloud** → **iCloudバックアップ** をタップ
3. 「このiPhoneをバックアップ」をオンにする
4. **「今すぐバックアップを作成」** で即時バックアップ可能

### 自動バックアップの条件（3つ同時に満たすと自動実行）
- 電源（充電ケーブル）に接続している
- Wi-Fiに接続している
- 画面がロックされている

## 削除した連絡先を復元する方法

iCloudは連絡先のスナップショット（アーカイブ）を自動保存しています。

### 手順
1. パソコンのブラウザで **icloud.com** にアクセス
2. Apple IDでサインイン
3. 右上のアカウントアイコン → **「アカウント設定」** をクリック
4. ページ下部の **「詳細設定」** セクションへスクロール
5. **「連絡先を復元...」** をクリック
6. 日付付きのアーカイブ一覧が表示されるので、削除前の日付を選択
7. **「復元」** をクリック

### 注意点
- 復元すると **現在の連絡先が上書き**される
- 復元前に現在の連絡先をエクスポート（.vcfファイル）しておくと安全
  - icloud.com の連絡先 → 全選択（⌘A）→ 書き出し

## iCloudの空き容量が足りない場合

無料プランは **5GB** のみ。容量が不足すると自動バックアップが止まる。

### 対処法
1. **不要なバックアップを削除**: 設定 → iCloud → ストレージを管理 → バックアップ
2. **写真をiCloud写真に移行**: バックアップとは別枠で管理できる
3. **iCloud+にアップグレード**: 50GB（月130円）、200GB（月400円）など

## LINEのバックアップ（別途設定が必要）

LINEのトーク履歴はiCloudバックアップに含まれないため個別設定が必要。

1. LINE → **設定** → **トーク** → **トークのバックアップ**
2. **「今すぐバックアップ」** をタップ
3. 定期バックアップを「毎日」に設定しておくと安心
