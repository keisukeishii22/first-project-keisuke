# CLAUDE.md

このプロジェクトは、石善建設(南房総の工務店・3代目 圭佑)の **note×Instagram 連動コンテンツ運用** を管理する。

## note×Instagram連動運用(2026年6月開始)

- 週次コンテンツは「12週テーママップ.md」に従って生成する。
- 「第N週のテーマパックを生成」と指示されたら、同ファイルの該当週の指定とインタビュー回答を読み込み、以下を一括出力する:
  1. note 記事下書き(タイトル案3本付き)
  2. リール台本2本(各30〜45秒。フック→本編→CTA の構成)
  3. カルーセル構成案(表紙+5枚+CTA)
  4. ストーリーズ告知文
- 同ファイル冒頭の「共通ルール」(PR 表記、現場写真禁止、ファクトチェック必須、固有名詞の扱い)を全生成物に適用する。
- 「情報源ガイドライン.md」を全生成物に適用する。note 記事には「参考情報源」欄を必ず付け、出典を示せない数字は書かない。Web 検索が使えるセッションでは、制度・法令・数字を扱う週は生成前に一次情報(国交省・e-Gov 等)を検索して裏取りする。
- [検索型]の週は、テーママップ共通ルールの「SEO ブロック」(想定検索キーワード・キーワード配置・FAQ 3問・ハッシュタグ案)を note 記事に必ず含める。
- 石善建設公式インスタ・圭佑不動産インスタ・石善建設 HP ブログのコンテンツを作るときは、`媒体別運用/` 配下の該当媒体のガードレール.md を必ず読み込んで適用する(不動産は宅建業法の広告規制があるため特に厳守)。
- 生成後、進捗メモ.md に「第N週パック生成済み(日付)」を追記する。
- 毎週の KPI(フォロワー数・保存数・プロフィールアクセス・note ビュー・LINE 登録数)を報告されたら、進捗メモ.md の KPI 記録欄に追記し、保存数上位テーマを翌週の生成に反映する。

## 自動運用ルーティーン

- 詳細は「自動運用ルーティーン.md」を参照。
- 毎週日曜、指示がなくても次の未生成の週のテーマパックを生成し、`生成物/第N週/` に保存する。
- インタビュー回答が未提供の週は、テーママップの指定内容のみで「素材待ちドラフト」を生成し、回答到着後に肉付けする方針とする。
- 生成物は必ず圭佑の確認(ファクトチェック・実体験の追記)を経てから投稿する。
