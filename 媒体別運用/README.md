# 媒体別運用

個人アカウント×note(ルートの12週テーママップで運用中)以外の媒体を、ここで管理する。

## 構成

| ディレクトリ | 媒体 | 状態 |
|--------------|------|------|
| `圭佑不動産インスタ/` | 圭佑不動産の Instagram | ガードレール策定済み。テーママップは物件情報・営業方針のヒアリング後に作成 |
| `石善建設インスタ/` | 石善建設の公式 Instagram | ガードレール策定済み。テーママップは施工事例の掲載許可運用の確認後に作成 |
| `石善建設HPブログ/` | 石善建設 HP のブログ | ガードレール策定済み。テーママップは既存 HP の構成確認後に作成 |

## 共通事項

- ルートの「情報源ガイドライン.md」は全媒体に適用する。
- 各媒体のコンテンツを生成するときは、必ずその媒体の `ガードレール.md` を読み込んでから生成する。
- 週次自動化を各媒体へ広げる場合は、個人アカウントと同じ型(テーママップ+進捗ログ+生成物ディレクトリ)を複製する。
