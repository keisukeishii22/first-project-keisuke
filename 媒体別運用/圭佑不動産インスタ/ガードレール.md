# 圭佑不動産 Instagram 運用ガードレール

不動産の広告・集客投稿は **宅建業法・不動産の表示に関する公正競争規約(表示規約)・景品表示法** の規制対象になる。工務店アカウントより一段厳しい前提で運用する。生成・投稿前に必ず本ファイルを適用する。

## 0. 大前提

- 物件を特定できる情報(所在・価格・間取り・写真など)を載せて反響を狙う投稿は、SNS でも「広告」として宅建業法・表示規約の適用を受ける。リール・ストーリーズ・ハイライトも同様。
- プロフィールに **商号・宅建業の免許証番号・所属団体(全宅連/全日等)・事務所所在地** を常設する。
- 迷ったら「物件広告」ではなく「知識・体験談コンテンツ」(相場の考え方、内見のチェックポイント、南房総の暮らし等)に倒す。知識系は広告規制の必要表示事項が不要で、炎上・違反リスクが低い。

## 1. 宅建業法の禁止・義務(条文ベース)

| 規制 | 内容 | 投稿での運用 |
|------|------|--------------|
| 誇大広告等の禁止(32条) | 著しく事実に相違する表示、実際より著しく優良・有利と誤認させる表示の禁止 | 「駅近」「格安」「掘り出し物」等は根拠なしに使わない。おとり広告(存在しない・取引意思のない・成約済み物件の掲載)は違反。**成約したら当日中に投稿を削除またはアーカイブ**し、ハイライトからも外す |
| 広告開始時期の制限(33条) | 建築確認・開発許可等の処分前は広告不可(新築の売買・貸借とも) | 未確認の新築・造成中物件は「チラ見せ」も含めて投稿しない |
| 取引態様の明示(34条) | 売主・代理・媒介の別を広告のたびに明示 | 物件投稿には毎回キャプション末尾に「取引態様:媒介」等を記載。ストーリーズはテキストで載せる |
| 断定的判断の提供の禁止(47条の2) | 利益が確実と誤解させる断定の禁止 | 「必ず値上がりします」「絶対に損しない」「利回り確実」は言い換え不可の NG ワード |

## 2. 表示規約(公正競争規約)の要点

- **徒歩表示**:道路距離 80m = 1分、端数は切り上げ。「駅から5分」は実測根拠を持つ。
- **物件広告の必要表示事項**:所在地・交通・面積・価格(賃料)・築年・取引態様など。Instagram の1投稿に収まらない場合は、**投稿には物件の魅力+取引態様のみ、詳細は HP/ポータルの物件ページへ誘導**し、リンク先で必要表示事項を満たす。
- **特定用語の使用基準**:「完全」「完璧」「日本一」「厳選」「特選」「格安」「掘り出し」等は、客観的根拠を示せない限り使用しない。
- **二重価格表示**:「○○万円→△△万円に値下げ!」は原則使わない(過去の価格の公表時期等、規約の条件を満たす場合のみ)。
- **写真・動画**:掲載するのは当該物件のもの。別物件やイメージ素材を使う場合は「イメージ」と明示。リフォーム済み表示は施工時期・内容を正確に。

## 3. 景品表示法・ステマ規制(2023年10月〜)

- 事業者アカウントによる自社物件・自社サービスの投稿は広告であることが分かる形にする。第三者(友人・インフルエンサー・お客様)に投稿を依頼する場合は **「#PR」等の明示が必須**。
- お客様の声・入居者インタビューは、実在・本人同意・改変なしの3条件を満たす。

## 4. プライバシー・撮影

- 入居中・居住中物件の内観は撮影・掲載しない(退去後または売主同意のある空室のみ)。
- 表札・郵便物・車のナンバー・近隣の人物が写り込んだ素材は使わない。
- 売主・貸主の同意なく物件を特定できる外観を出さない。

## 5. 投稿前チェックリスト(毎回)

1. [ ] これは「物件広告」か「知識コンテンツ」か判別した
2. [ ] (物件広告の場合)取引態様を明示した/建築確認等の処分済みを確認した/成約時に消す運用を担当者と確認した
3. [ ] 徒歩分数・価格・面積など数字の根拠資料がある
4. [ ] NG ワード(必ず・絶対・完全・格安・掘り出し 等)を含まない
5. [ ] 写真は当該物件・掲載同意あり・写り込みなし
6. [ ] プロフィールの免許番号等の表示が最新
7. [ ] 数字・制度に触れる場合は「情報源ガイドライン.md」に従い一次情報で裏取りした
