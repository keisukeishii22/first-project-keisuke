# 石善建設 公式 Instagram 運用ガードレール

会社の看板を背負うアカウント。個人アカウント(圭佑)より「正確さ・お客様保護」を優先する。

## 1. 施工事例・現場の掲載

- 施工事例は **施主の書面またはメッセージでの掲載同意** を得たものだけ掲載する(同意の記録を残す)。
- 住所・外観全景・表札など、家が特定できる情報は同意があっても最小限にする。掲載範囲(内観のみ/外観可/エリア表記の粒度)を施主ごとに決める。
- 工事中の現場は、安全帯・ヘルメット等の保護具が正しく写っているカットのみ使う(不安全行動が写った素材は法令順守姿勢を疑われるため不使用)。
- 職人・協力業者が写る場合は本人の同意を得る。

## 2. 品質・性能の表現

- 「耐震等級3」「断熱等級○」「長期優良住宅」等の性能表示は、**該当物件で実際に取得・証明できるものだけ**書く。会社としての標準仕様と個別物件の実績を混同しない。
- 「地震に絶対強い」「一生メンテナンス不要」など保証できない断定表現は使わない。
- 建築基準法・品確法・省エネ基準に触れる投稿は「情報源ガイドライン.md」に従い一次情報(国交省)で裏取りする。

## 3. 価格・補助金

- 価格表示は条件(本体のみ/付帯込み、税込・税抜、坪単価の算定条件)を必ず添える。条件なしの「○○万円で建つ」は出さない。
- 補助金・減税は年度・予算枠で変わるため、**申請期限と「最新情報は公式サイトで」の一文を必ず入れる**。終了した制度の投稿はアーカイブする。

## 4. 取引・営業上の配慮

- 元請(ハウスメーカー)案件の現場・仕様は投稿しない(守秘条項)。自社元請案件のみ。
- 他社(ハウスメーカー・同業)批判はしない。比較は「仕組みの解説」に徹する。
- お客様の声は実在・本人同意・原文尊重。謝礼を出した場合はその旨を明示(ステマ規制)。

## 5. 投稿前チェックリスト

3. [ ] 性能・価格・補助金の数字に根拠と条件を付けた
4. [ ] 断定・保証表現(絶対・必ず・一生 等)がない
5. [ ] 元請案件・守秘対象が写っていない
